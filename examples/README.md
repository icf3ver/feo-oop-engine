# Examples
To run an example use the following command.

```terminal
cargo run --bin <example>
```

## Structure
Most examples use the following layout
```terminal
│   main.rs
└───scr
        mod.rs
        <scripts>
```
scr is short for script and is where I have chosen to store the scripts.
